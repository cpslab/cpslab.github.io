CPS Lab homepage

# Core

* jekyll
* ruby 2.4.1

# Contributes

## ruby環境構築


```
brew update
brew upgrade ruby-build
rbenv install 2.4.1
```

## 環境構築


```
gem install bundler
bundle install
jekyll serve
```

Open [http://localhost:4000](http://localhost:4000)
